# 📦 Inventory Management System

A simple and responsive **Inventory Management System** built using **HTML, CSS, and JavaScript**.

This project is designed for learning web development and as a university project. It provides a clean dashboard UI for managing inventory information.

---

## 🚀 Features

- 🏠 Dashboard
- ➕ Add Product
- 📊 Total Products
- 📦 Total Stock
- 💰 Total Inventory Value
- ⚠️ Low Stock Counter
- 🎨 Responsive User Interface
- 📱 Mobile Friendly Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **home_page.html** or open it in your web browser.

No installation or backend is required.
